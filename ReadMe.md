    ```

    ├── <tarball>.tar.gz
    ├── <tarball>.tar.gz
    ├── <tarball>.tar.gz

    ├── csc254_a1.c
    ├── csc254_a1

    └── testcases

    ```
- specify testcases in `./testcases`
- `./compile.sh`
- There should be RESULT files generated for grading
